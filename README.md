# Multiplication Project

Mon premier projet en JavaScript permettant de s’entraîner aux tables de multiplication de manière interactive.

## Objectif

Ce projet a pour but de :

* pratiquer les bases de JavaScript
* manipuler le DOM
* gérer des événements utilisateur
* créer une petite application interactive

## Fonctionnalités

* Génération aléatoire de multiplications
* Saisie de la réponse par l’utilisateur
* Vérification de la réponse
* Feedback (bonne ou mauvaise réponse)
* Affichage de score dynamique
* Vitesse de réponse

## Technologies utilisées

* HTML
* CSS
* JavaScript

## Ce que j’ai appris

* Utiliser `addEventListener`
* Modifier le DOM dynamiquement
* Gérer des variables et des conditions
* Créer une logique simple d’application

## Améliorations possibles

* Ajouter plusieurs niveaux de difficulté
* Sauvegarder le score
* Rendre le design plus interactif

## Auteur

Projet réalisé dans le cadre d’un apprentissage du JavaScript avant formation.
